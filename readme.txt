== Description ==

Admin Icons Manager is a management plugin for you edit the icons of the Dashboard menu or Wordpress admin menu. You can change its color or put another icon (dashicons or font awesome or image).

You can manager:
> * Icons color
> * Icons font
> * Images for icons

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/plugin-name' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Admin Icons Manager screen to configure the plugin
